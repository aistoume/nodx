Install nodx Lens on macOS (beta)

Unsigned beta — takes 60 seconds + one terminal command.

Update: the standalone Lens-for-Mac app has been merged into nodx desktop (since 0.3). Download nodx desktop below to get system-wide ⌥+E select-and-explain plus the full thinking workspace — the install steps on this page apply the same way.
⬇ Download nodx-0.5.0-arm64.dmg

Apple Silicon · macOS 12+ · ~8 MB

1. Install

  1. Double-click the downloaded .dmg
  2. Drag nodx to the Applications folder
  3. Eject the DMG

2. Bypass Gatekeeper (one-time)

Why this step exists: macOS marks any app downloaded from the internet with a "quarantine" flag. Apps that aren't signed by Apple Developer Program (we'll be properly signed after the beta) trigger a fake "nodx Lens is damaged and can't be opened" message. It's not actually damaged — Apple just refuses to open it until you remove the flag.

Open Terminal (⌘+Space → "Terminal") and paste:

xattr -dr com.apple.quarantine "/Applications/nodx.app"

That's it. Double-click nodx Lens in Applications — it opens normally.

3. Grant Accessibility permission

nodx Lens needs Accessibility to simulate ⌘+C when you press ⌥+E. On first launch, press ⌥+E:

  1. A "Permission Needed" window appears
  2. System Settings auto-opens to Privacy & Security → Accessibility
  3. Toggle nodx Lens on
  4. From the menu bar (top-right) → click the nodx Lens icon → Quit
  5. Re-launch nodx Lens from Applications

4. Configure your API key

Press ⌥+E — Settings opens.

  • Provider — Anthropic, OpenAI, or Google
  • API Key — from your chosen provider's console
  • Model — defaults are fine
  • Language — English or 中文

5. Use it

  1. Select text in any app (Notes, VS Code, Safari, PDF...)
  2. Press ⌥+E
  3. nodx Lens captures your selection and shows an AI explanation
  4. Press Esc to dismiss

What it does (and doesn't)

  • ✅ Reads the text you just selected (simulates ⌘+C)
  • ✅ Sends it directly to your AI provider with your own key
  • ✅ Streams the response into a small floating window
  • ✅ Restores your original clipboard so we don't pollute what you had copied
  • ❌ No background telemetry. No analytics. No phone-home.
  • ❌ No reading anything outside the ⌥+E trigger.
  • ❌ No nodx server in the loop — your data goes direct to Anthropic / OpenAI / Google.

Uninstall

  1. Finder → Applications → drag nodx Lens to Trash
  2. (Optional) Clear settings: rm -rf ~/Library/Application\ Support/com.nodx.lens
  3. (Optional) Remove Accessibility entry: System Settings → Privacy → Accessibility → select nodx Lens → click −

Troubleshooting

  • "⌥+E does nothing" — check Accessibility permission is on AND restart nodx Lens after granting
  • "Window appears in the wrong spot" — V0 centers the window; V1 will spawn at cursor
  • "Error: 401 / 403" — API key wrong / out of quota / wrong model name
