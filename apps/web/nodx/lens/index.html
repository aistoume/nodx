Six things other "AI explainers" don't do

💾 Save as thinking card

One click on any explanation packages it as a thinking card (snippet + AI explanation + source) into your local library. nodx desktop picks these up as starting points for structured reasoning — the "attention token" that turns reading into deciding.

📌 Persistent annotations

The blue underline survives panel close. Click it later — same explanation re-opens, no second API call. Like Grammarly's underlines, but for understanding instead of grammar.

🔍 Select-to-explain

Highlight 2–500 characters → click 🔍 → streamed explanation in place. 50–150 words by default; tap "Deepen" for a 200–400 word version with examples.

🌊 Real streaming

Words appear as the model generates them. Judge within the first sentence whether the answer is on-track — close early if it isn't.

🔐 Local-first, zero telemetry

No nodx server. API key, settings, history, snippets all in chrome.storage.local. Requests go directly from your browser to your AI provider.

🧰 Truly free + BYO key

Anthropic Claude, OpenAI GPT, or Google Gemini — pick your model, pay only for what you use. No subscription, no Pro tier, no usage caps.

From reading to thinking — in 30 seconds

  1. Install from the Chrome Web Store (60 seconds).
  2. Open Settings — pick a provider, paste your own API key.
  3. Highlight any phrase on any webpage — the 🔍 trigger floats above.
  4. Click → read — explanation streams in; underline persists.
  5. Save to nodx — explanation becomes a thinking card you can revisit, or forward to nodx desktop when it ships.
