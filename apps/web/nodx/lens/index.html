One action wheel, two ways in

🎡 The action wheel — now fully yours

Four spokes, every one customizable: icon (40-emoji library or your own image), name, colour, action and prompt. Turn any spoke into a submenu. Search destinations come from a preset library — 21 sites: Google, Amazon, Temu, Taobao, JD, arXiv, GitHub, YouTube… — or paste your own URL. Text and screenshots share the same wheel.

📦 Boxes are action hubs

Every capture leaves a yellow box on the page. Click it to reopen the wheel and act on the same capture again. Its corner badge tells the story — 💬 3 🔎 🎨 means three Q&A turns, a search, and a generated image. One region never grows duplicate boxes.

🎤 Say it, done (Android)

The wheel's Voice spoke goes straight to speech: say "open it on YouTube" or "explain it to me" and it just runs — no typing, auto-executed, same intent routing. Uses the system recognizer, so it speaks your language and needs no extra permission.

✏️ Instructions that act

Hit the Instruct spoke and type what you want — "translate this", "extract the dates", "search it on arXiv". Search-type instructions open the real results page; answers stream into a panel you can keep talking to, and every run lands in history.

👁 Vision on any region

Box a product photo, a chart, a UI — ask "what is this?" about the exact pixels. Visual search uses Google Lens for real images, AI recognition + image search for everything else.

🎨 Image generation

From selected text or a boxed region, generate one image laid out as a 2×2 style grid — product photo, watercolor, 3D render, line art — powered by Gemini's image model with your own key.

🗂 The side panel remembers

Current-page captures on top, each with its own Q&A thread. A collapsible history logs every search, shopping trip, and generated image with a re-open link. Underlines and boxes persist across sessions.

🔐 Local-first · 5 providers, one free

Claude, GPT, Gemini, OpenRouter's free models (no cost, just a free key), or your local nodx gateway via Claude Code — each with its own saved key. Quick-switch models right in the side panel. No nodx server, no telemetry, no subscription.

From seeing to acting — in 30 seconds

  1. Install from the Chrome Web Store (60 seconds).
  2. Open Settings — pick a provider, paste your own API key.
  3. Select text on any page, or click 📸 Screenshot region in the side panel and drag over anything.
  4. Pick a spoke — explain it, search it, shop it, or generate from it.
  5. Come back anytime — the box stays on the page; click it to keep going, or send it to nodx desktop.

Run it on YOUR local Claude (no API key)

Have Claude Code installed and logged in? Lens can use it directly — no key pasted anywhere, nothing leaves your machine.

  1. Gateway mode (keep a terminal open): npx nodx-lens-gateway, then Settings → AI Provider → nodx (local gateway).
  2. Native mode (zero terminal): copy the one-line installer shown in Settings → "Direct local Claude", run it once, click Connect local Claude. Chrome then spawns Claude on demand.
